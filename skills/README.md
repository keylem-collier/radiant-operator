# Frontend Motion Skills

Local Codex skills for clean micro-animation, micro-motion, bouncy spring feel, fluid UI transitions, and richer interactive motion assets.

These were selected from current web research on frontend UI motion patterns and tooling:

- Motion for React for component state, gesture, spring, and layout animation.
- GSAP with ScrollTrigger for timeline choreography and scroll-linked motion.
- Native CSS and the View Transition API for lightweight transitions.
- Rive and LottieFiles for authored interactive or vector animation assets.
- Accessibility and performance guidance from MDN and web.dev.

## Skills

- `clean-micro-animation`
- `bouncy-spring-motion`
- `fluid-ui-motion`
- `gsap-scroll-choreography`
- `interactive-motion-assets`

Use the smallest skill that fits the work. Combine skills only when the UI genuinely needs multiple motion systems.
